Predict a student's math score based on various factors using machine learning.

About This Project

This application uses machine learning to predict student math scores based on demographic information and other test scores. The model was trained on student performance data and can help educators identify factors that influence academic performance.

API Access

This service is also available via REST API. Send a POST request to /api/predict with JSON data containing student information.
